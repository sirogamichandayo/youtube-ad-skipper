### 概要

YouTubeの高校を自動的にスキップしてくれるchrome拡張

### できること
* 動画のスキップ(beta)

### 使い方

リポジトリをクローンする
chrome拡張のでペロッパーモードをオンにする
パッケージ化されていない拡張機能を読み込むを選択して、`skipper`ディレクトリを選択する
拡張機能をonにする
